User Story 1 (US-1): Book Registration

As a librarian,
I want to add and store books with unique Book ID, Title, and Author,
So that the library maintains correct book records.

User Story 2 (US-2): Borrow Book

As a library member,
I want to borrow an available book,
So that I can read it.

User Story 3 (US-3): Return Book

As a library member,
I want to return a borrowed book,
So that it becomes available for others.

User Story 4 (US-4): Library Report

As a librarian,
I want to generate a library status report,
So that I can view Book ID, Title, Author, and Availability.
